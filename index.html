BP interface can communicate with sublevel but not with persistent level.

  1. create new sublevel name "ScriptSubLevel"
  2. make sure the streaming method is always loaded
  3. in iInteractable make new function LevelComm that will return a bool
  4. edit the sublevel blueprint and implement interface iInteractable
